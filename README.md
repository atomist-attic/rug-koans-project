# Atomist 'rug-koans-project'

***Warning!*** - Working in Progress.

A generator for the starting point in your journey through a set of learning tutorials, [koans][koans], for getting to grips with [Rug][rug]. This starting point is then used to executor editors from [Rug Koans Editors][rug-koans-editors] to work through each of the Koans.

[rug]: http://docs.atomist.com/
[koans]: (https://en.wikipedia.org/wiki/K%C5%8Dan)
[rug-koans-editors]: https://github.com/atomist-rugs/rug-koans-editors
